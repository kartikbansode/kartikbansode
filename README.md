## 🚀 What I'm Building

### **FinWise India** — Financial OS for Freelancers
A comprehensive financial management platform designed for Indian freelancers, consultants, and small business owners.

- 📊 GST Management & Invoicing
- 💰 Tax Calculation (44ADA/44AD sections)
- ⏰ Advance Tax Reminders & Alerts
- 📈 Financial Dashboard & Analytics

**Tech Stack:** Next.js • Supabase • TypeScript • Framer Motion • Tailwind CSS

**Status:** Core app functional | Landing page in progress (Apple-style design with 3D effects)
